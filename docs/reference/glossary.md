## 変電

| 用語 | 読み | 意味 | 関連テーマ |
|------|------|------|-----------|
| 変圧器 | へんあつき | 電圧を変換する機器。1次・2次の巻き数比で電圧が決まる | 変圧器 |
| 鉄損 | てつそん | 磁束の変化により鉄心に生じる損失（ヒステリシス損＋渦電流損）。負荷に無関係 | 変圧器 |
| 銅損 | どうそん | 電流が巻き線の抵抗を流れることで生じる損失。負荷電流の2乗に比例 | 変圧器 |
| 最大効率 | さいだいこうりつ | 鉄損＝銅損となる負荷時に変圧器効率が最大になる | 変圧器 |
| パーセントインピーダンス | ぱーせんといんぴーだんす | 定格電流を流したとき変圧器の内部インピーダンスによる電圧降下の定格電圧に対する比（%） | 変圧器 |
| 遮断器（CB） | しゃだんき | 事故時に回路を遮断する機器。アーク消弧能力が必要 | 開閉装置 |
| 断路器（DS） | だんろき | 無電圧・無負荷状態で回路を開閉する機器。負荷電流の遮断不可 | 開閉装置 |
| 過電流継電器（OCR） | かでんりゅうけいでんき | 設定値以上の電流が流れると動作して遮断器を制御する継電器 | 保護継電器 |
| 地絡継電器（GR） | ちらくけいでんき | 地絡事故を検出する継電器 | 保護継電器 |
| GIS | じーあいえす | ガス絶縁開閉装置。SF6ガスで絶縁・アーク消弧。小型化が可能 | 開閉装置 |

## 送電

| 用語 | 読み | 意味 | 関連テーマ |
|------|------|------|-----------|
| 電圧降下 | でんあつこうか | 送電線のインピーダンスによる送電端と受電端の電圧差 | 架空送電線路 |
| 電力損失 | でんりょくそんしつ | 送電線の抵抗による発熱損失 P=3I²R（三相）または I²R（単相） | 架空送電線路 |
| 力率改善 | りきりつかいぜん | 進相コンデンサ等で力率を1に近づけ、無効電力を減らすこと | 架空送電線路 |
| ACSR | えーしーえすあーる | 鋼心アルミより線。軽量で引張強度が高い送電線用電線 | 架空送電線路 |
| 弛度（たるみ） | しど | 電線が自重でたわんだ量。温度上昇や着雪で増加 | 架空送電線路 |
| 実効値 | じっこうち | 交流の実効的な大きさ。最大値の1/√2 | 架空送電線路 |
| スキン効果 | すきんこうか | 高周波では電流が導体表面に集中する現象 | 架空送電線路 |
| フェランチ効果 | ふぇらんちこうか | 軽負荷・無負荷時に受電端電圧が送電端電圧より高くなる現象 | 架空送電線路 |
| XLPE | えっくすえるぴーいー | 架橋ポリエチレン絶縁。地中ケーブルの主流。耐熱性・絶縁性に優れる | 地中送電 |
| OFケーブル | おいるふぃるどけーぶる | 絶縁油を充填した油浸紙絶縁ケーブル。高電圧対応 | 地中送電 |

|------|------|------|-----------|

| 低圧バンキング方式 | ていあつばんきんぐほうしき | 同一高圧線に接続した複数の変圧器の低圧側を並列接続 | 配電線路 |

| 進相コンデンサ | しんそうこんでんさ | 遅れ無効電力を補償して力率を改善するコンデンサ | 配電線路 |
| 変圧器タップ | へんあつきたっぷ | 変圧器の巻き数を切り替えて電圧を調整する端子 | 配電線路 |

## 電力系統・需給

| 用語 | 読み | 意味 | 関連テーマ |
|------|------|------|-----------|
| 周波数調整 | しゅうはすうちょうせい | 発電と需要のバランスを保ち系統周波数を50/60Hzに維持すること | 電力系統 |
| 負荷周波数制御（LFC） | ふかしゅうはすうせいぎょ | 系統周波数のズレを検出して発電機出力を自動制御する | 電力系統 |
| 電力系統安定度 | でんりょくけいとうあんていど | 擾乱後も同期運転を維持できる能力（定態・過渡・動態安定度） | 電力系統 |
| 需給バランス | じゅきゅうばらんす | 発電量と消費量を常時一致させること（電気は貯蔵できない） | 電力系統 |
| ピーク負荷 | ぴーくふか | 1日・1年の最大需要電力。ピーク対応に揚水発電等を活用 | 電力系統 |
| 自動周波数制御（AFC） | じどうしゅうはすうせいぎょ | 電力系統の周波数・潮流を自動的に制御するシステム | 電力系統 |
| SCADA | すかだ | 広域での発電・送電設備を監視・制御するシステム | 電力系統 |
| 系統連系 | けいとうれんけい | 太陽光等の分散型電源を電力系統に接続すること | 電力系統 |
