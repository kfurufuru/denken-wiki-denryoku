# 絵文字マーカー凡例

| 絵文字 | 意味 | 使用場面 |
|--------|------|---------|
| 🧠 | 直感的理解・アナロジー | 「水流＝電流」のような比喩で掴む |
| 🏭 | 設備を歩く（設備構成） | 設備フロー図・主要機器の役割 |
| ⚖️ | 比較表 | 水車の種類比較、熱サイクル比較 |
| 🔍 | 公式の意味マップ | 公式の意味・使える条件 |
| 🛤️ | 解法パターン | 計算問題の「型」 |
| 🕳️ | 落とし穴・頻出ミス | 間違えやすいポイント |
| 📝 | 正誤判定の急所 | 正誤問題でよくある誤文パターン |
| 🎯 | 重要ポイント | 特に押さえるべき事項 |
| ⚡ | 5秒で思い出す | 速攻暗記フレーズ |
| 📊 | 出題実績 | 年度別の出題記録 |
| 😱💀🔥💥 | 事故・危険 | 安全に関わる重大事項 |
| ✅ | 確認済み | 教科書と照合済み |
| ⚠️ | 未確認 | 要検証 |
