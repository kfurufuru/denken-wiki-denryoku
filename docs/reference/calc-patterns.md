# 頻出計算パターン集

電力科目で繰り返し出題される計算の「型」を5パターンで整理。

!!! tip "使い方"
    各パターンの「公式」を覚え、「手順」通りに当てはめるだけで解ける。
    「注意点」は落とし穴なので必ず確認すること。

---

## パターン1: 水力発電の出力計算

### 公式

```
P = 9.8 × Q × H × η  [kW]
```

| 記号 | 意味 | 単位 |
|------|------|------|
| P | 発電出力 | kW |
| Q | 流量 | m³/s |
| H | 有効落差 | m |
| η | 総合効率（水車効率×発電機効率） | — |

### 手順

1. 有効落差 H を確認（総落差 - 損失水頭）
2. 流量 Q を確認
3. 効率 η が水車効率×発電機効率で与えられていれば掛け合わせる
4. P = 9.8QHη に代入

### 注意点

!!! danger "落とし穴"
    - **9.8** は重力加速度（g=9.8 m/s²）× 水の密度（1000 kg/m³）× 単位換算の結果。暗記必須
    - 有効落差（H）であることに注意。総落差を使うと誤答
    - 出力の単位は **kW**（Wではない）

### 例題イメージ

> 有効落差 100 m、流量 10 m³/s、効率 0.85 の水力発電所の出力は?
>
> P = 9.8 × 10 × 100 × 0.85 = **8,330 kW**

## パターン2: 火力発電の熱効率計算

### 公式

```
η = 3600 / H  × 100  [%]
```

または

```
η = P_out / Q_in  [小数]
```

| 記号 | 意味 | 単位 |
|------|------|------|
| η | 発電端熱効率 | % |
| H | 熱消費率 | kJ/kWh |
| P_out | 発電電力量 | kWh |
| Q_in | 投入熱量 | kJ |

### 手順（熱消費率から求める場合）

1. 熱消費率 H [kJ/kWh] を確認
2. η = 3600 / H × 100 [%] に代入

### 手順（燃料消費量から求める場合）

1. 発電電力量 = 出力 × 時間
2. 燃料消費量 × 低位発熱量 = 投入熱量
3. η = 発電電力量 [kJ] / 投入熱量 [kJ]

### 注意点

!!! danger "落とし穴"
    - 3600 は 1 kWh = 3600 kJ の換算係数
    - **低位発熱量**を使う（高位発熱量と混同しない）
    - 熱消費率が小さいほど効率が良い（逆数の関係）

### 例題イメージ

> 熱消費率 9000 kJ/kWh の火力発電所の熱効率は?
>
> η = 3600 / 9000 × 100 = **40%**

## パターン3: 変圧器の効率計算

### 公式

```
η = P_out / (P_out + Pi + Pc)
```

または負荷率 m を使って:

```
η = m × P_n × cosφ / (m × P_n × cosφ + Pi + m² × Pc_n)
```

| 記号 | 意味 |
|------|------|
| P_out | 出力（有効電力）= P_n × m × cosφ |
| Pi | 鉄損（一定）|
| Pc | 銅損（負荷電流の2乗に比例）|
| P_n | 定格容量 |
| m | 負荷率（実負荷/定格）|
| Pc_n | 定格時の銅損 |

### 手順

1. 負荷率 m を確認
2. 出力 = m × P_n × cosφ を計算
3. 銅損 = m² × Pc_n（負荷率の2乗に比例することに注意）
4. η = 出力 / (出力 + 鉄損 + 銅損)

### 注意点

!!! danger "落とし穴"
    - **鉄損は一定**、銅損は電流の2乗（≒負荷率の2乗）に比例
    - 最大効率条件: **鉄損 = 銅損**（最大効率負荷率 = √(Pi/Pc_n)）
    - 全日効率は「1日の出力電力量 / 1日の入力電力量」で計算

### 例題イメージ

> 100 kVA、鉄損 500 W、定格銅損 1000 W、力率 0.8、負荷率 0.5 のとき効率は?
>
> 出力 = 0.5 × 100 × 0.8 = 40 kW
> 銅損 = 0.5² × 1000 = 250 W
> η = 40000 / (40000 + 500 + 250) = **98.2%**

## パターン4: 送電線の電圧降下計算

### 公式

三相3線式:

```
e ≈ √3 × I × (R cosθ + X sinθ)  [V]
```

単相2線式:

```
e ≈ 2 × I × (R cosθ + X sinθ)  [V]
```

| 記号 | 意味 | 単位 |
|------|------|------|
| e | 電圧降下（送電端-受電端）| V |
| I | 線電流 | A |
| R | 1線の抵抗 | Ω |
| X | 1線のリアクタンス | Ω |
| cosθ | 力率 | — |
| sinθ | 無効率 = √(1-cos²θ) | — |

### 手順

1. 力率 cosθ から sinθ = √(1-cos²θ) を計算
2. 線電流 I を電力から算出（三相: I = P / (√3 × V × cosθ)）
3. 公式に代入

### 注意点

!!! danger "落とし穴"
    - 三相は **√3 倍**、単相2線は **2倍**（単相3線は配線の状況次第）
    - R, X は **1線分**の値（三相全体ではない）
    - 電圧降下率 = e / V_s × 100 [%] で表す場合もある

### 例題イメージ

> 三相3線式、R=2Ω、X=3Ω、I=100A、cosθ=0.8（sinθ=0.6）のとき電圧降下は?
>
> e = √3 × 100 × (2×0.8 + 3×0.6) = 1.732 × 100 × 3.4 = **589 V**

## パターン5: 配電線の電力損失計算

### 公式

三相3線式:

```
P_loss = 3 × I² × R  [W]
```

単相2線式:

```
P_loss = 2 × I² × R  [W]
```

| 記号 | 意味 | 単位 |
|------|------|------|
| P_loss | 電力損失 | W |
| I | 線電流 | A |
| R | 1線の抵抗 | Ω |

### 手順

1. 線電流 I を確認（電力・電圧・力率から計算）
2. 公式に代入

### 注意点

!!! danger "落とし穴"
    - 三相は **3本**の線があるので係数は3
    - 電流が2倍になると損失は **4倍**になる（電流の2乗に比例）
    - **力率改善の効果**: cosθ が上がると電流 I が減り損失が大幅減
    - 電圧を2倍にすると電流は半分になり損失は **1/4** になる

### 力率改善前後の損失比較

```
P_loss比 = (cosθ₁ / cosθ₂)²
```

> 力率 0.6 → 0.9 に改善すると損失は (0.6/0.9)² = **0.44倍**（56%削減）

### 例題イメージ

> 三相3線式、R=1Ω/線、負荷 100 kW、線間電圧 6600 V、cosθ=0.8 のとき電力損失は?
>
> I = 100000 / (√3 × 6600 × 0.8) = 10.95 A
> P_loss = 3 × 10.95² × 1 = **360 W**
