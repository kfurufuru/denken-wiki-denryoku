## 凡例・補足

### テーマ略称
| 略称 | テーマ名 | 主な出題問番号 |
|------|---------|------------|
| 水 | 水力発電 | 問1〜2, 問15 |
| 火 | 火力発電 | 問2〜3, 問15 |
| 原 | 原子力発電 | 問4 |
| 新 | 新エネルギー | 問5 |
| 変 | 変圧器・変電設備 | 問6〜7, 問16 |
| 開 | 開閉装置・保護継電器 | 問8 |
| 架 | 架空送電 | 問7〜9, 問11 |
| 地 | 地中送電 | 問9 |
| 配 | 配電線路 | 問7, 問10〜13, 問16〜17 |
| 系 | 電力系統・需給運用 | 問10 |
| 材 | 電気材料 | 問14 |

### 問題タイプ
- **論説**: 空欄なし・論述形式（選択問題含む）
- **穴埋**: 文中の空欄に語句・数値を埋める形式
- **計算**: 数値計算を要する形式

### 注意事項
- R04上以前の年度は年1回試験（上期/下期の区別なし）
- R05〜以降は年2回試験（上期=CBT方式、下期=筆記方式）
- 「—」は当該テーマからの出題実績データなし（他テーマが出題されている可能性あり）
- 1セルに複数テーマ記載は「同問番号に複数テーマ記事が記録していた場合」（テーマ記事の分類の重複による）
