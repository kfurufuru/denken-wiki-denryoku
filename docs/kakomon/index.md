# 過去問マッピング

電力科目の過去問を年度別・分野別で索引化。

| 索引 | 内容 |
|------|------|
| [年度別一覧](by-year.md) | H18〜R7下期の問題を年度順に配列 |
| [分野別一覧](by-field.md) | テーマごとに過去問を集約 |

---

## 使い方

- **弱点強化**: 分野別一覧 → 苦手テーマの問題を集中演習
- **年度別演習**: 年度別一覧 → 直近3年を通し演習
- **難易度確認**: 各問の難易度★で優先順位をつける

> ⚠️ 過去問データは順次充填中。電験王（denken-ou.com）で詳細解説を参照。
